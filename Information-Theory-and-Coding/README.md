# Information Theory & Coding Systems

This folder contains computational implementations focused on Information Theory, error-detecting codes, and combinatorial discrete coding structures.

##  Implemented Methodology
* **Hamming Distance Metric:** Implementing algorithmic loops to calculate individual coordinate shifts (bit transformations) between binary vectors.
* **Minimum Distance Isolation ($d_{min}$):** Evaluating complete generated codebooks to dynamically identify the minimal Hamming distance profile, which determines the error-detecting and error-correcting capabilities of the code.
* **Parity-Check & Block Code Generation:** Structuring systematic binary configurations to monitor data transmission integrity.
* **Constant-Weight Coding:** Generating subset matrices where each operational binary vector contains an exact pre-defined number of active bits (constant weight $w$), and computing the system capacity metrics.

##  Technologies Used
* **Python 3**
* **Math Module:** For discrete combinatorics and analytical capacity estimations.
