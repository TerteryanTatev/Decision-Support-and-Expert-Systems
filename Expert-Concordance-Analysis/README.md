# Decision Support and Expert Systems

Welcome to the **Decision Support and Expert Systems** repository. This portfolio contains academic projects, algorithmic implementations, and statistical tools designed to model expert consensus, optimize multi-criteria decision-making tasks, and evaluate ranking structures.

##  Implemented Projects

* **`Expert-Concordance-Analysis/`**
  * An expert consensus validation framework utilizing Kendall's W concordance metrics and $\chi^2$ hypothesis testing to evaluate the consistency of multi-expert ranking data.

##  Core Technologies
* **Python 3**
* **NumPy:** For processing tabular rank distributions and matrix variations.
* **Pandas:** For structural mapping of expert criteria and automated metrics summary logging.
